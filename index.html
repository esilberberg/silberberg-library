Silberberg Library

What is the Silberberg Library?

The Silberberg Library is the family's union catalog of books and audio recordings. Inquire if there is an item you would like to borrow.

The site uses Google Sheets as its database and takes advantage of the Sheets API to deliver all library data. Filtering is performed in the browser. The underlying application is written in vanilla JavaScript, making it lightweight, free, open source, and easily maintainable. This platform serves as a practical solution for those who have spreadsheet-based projects and are seeking for a way to dynamically share their work with the world.

Further details about the project can be found in the GitHub repo.
